# guosh

## Installing

```bash
$ git clone https://github.com/handicraftsman/guosh/ -b v0.4.2
$ cd guosh
$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
```

## dumbcpm

This package is also available as a dumbcpm package (`guosh`).

Library target is called `guosh/guosh`.
